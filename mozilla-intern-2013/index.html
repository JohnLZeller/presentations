What is Release Engineering?

Overview of Mozilla


• Define where RelEng fits under Mozilla umbrella

Overview of Mozilla


• Many groups, RelEng is one… 18 full-time, 4 interns
• 4 non-adjacent time zones, one of the more dispersed groups

Put Another Way

Release Engineering encompasses a part of the process between when a Developer makes a software check-in and when that change is ready to ship out to the user.

Release Engineers != Developers

Developers build a product

Release Engineers build a pipeline

So what does this Pipeline look like?

Nagios Alerts

Nagios Alerts



• We use Nagios extensively to monitor all aspects of the pipeline
• My alert had to do with pending builds/tests

Nagios Alerts


• Between when a Dev makes a check-in and buildbot grabbing that change, builds/tests are pending before going to machines in pool

Nagios Alerts

Nagios Alerts

Nagios Alerts
